
## Description

Microservice for training

## Instructions

npm install

npm run start

## Running the app

First run 

```npm install```

Then: 

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
